# Integration Tests

This directory contains integration tests for your frontend-app.

## Getting Started

1. Review `example.test.js` for patterns specific to frontend-app projects
2. Rename and customize the example test for your use case
3. Run integration tests: `npm run test:integration`

## Test Types for frontend-app


- **Unit Tests**: Components, utilities, hooks
- **Integration Tests**: Component interactions, forms
- **E2E Tests**: Browser automation, user flows
- **Accessibility Tests**: Screen reader, keyboard navigation
